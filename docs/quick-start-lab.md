## 🔌 简化接线图

```
ESP32 引脚分配（简化版）
├─ GPIO34 → 土壤湿度传感器
├─ GPIO4  → DHT22温湿度
├─ GPIO12 → 继电器IN
├─ 3V3    → 传感器VCC
├─ GND    → 公共地线
└─ USB    → 5V供电

继电器输出：
├─ COM → 5V（来自USB）
├─ NO  → 水泵正极
└─ 水泵负极 → GND
```

### 最简接线步骤

1. **土壤传感器** → GPIO34、3V3、GND
2. **DHT22** → GPIO4、3V3、GND
3. **继电器** → GPIO12、5V、GND
4. **水泵** → 接到继电器输出端

## ⚡ 快速组装（1小时）

### Step 1: 面包板测试（15分钟）
```
1. ESP32连电脑USB
2. 上传测试代码
3. 测试传感器读数
4. 测试继电器工作
```

### Step 2: 培育箱准备（15分钟）
```
1. 找一个废旧储物箱（20-30L）
2. 盖子打2个孔（传感器、水管）
3. 放入2个种植盆
4. 装入营养土
```

### Step 3: 播种（10分钟）
```
1. 种植盆挖坑（3cm深）
2. 放入土豆块（芽眼朝上）
3. 盖土压实
4. 浇透水
5. 插入土壤传感器
```

### Step 4: 启动系统（20分钟）
```
1. 连接所有组件
2. ESP32上电
3. 配置WiFi
4. 打开控制面板
5. 设置自动浇水
```

## ⚠️ 实验注意事项

### 安全第一
- 水泵用电安全（5V较安全）
- 避免水源溅到ESP32
- 定期检查线路

### 预期失败点
1. **传感器不准** → 电容式比电阻式好
2. **水泵堵塞** → 用纯水，加滤网
3. **WiFi断连** → 确保信号强度
4. **土豆腐烂** → 切块后必须晾干
5. **浇水过多** → 设置合理的阈值

### 失败了怎么办
- 土豆坏了 → 重新切种薯
- 电路坏了 → 检查接线
- 代码问题 → 查看调试日志
- **实验允许失败，重点是学习**
