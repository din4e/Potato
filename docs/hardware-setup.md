## 接线参考

```
ESP32 接线:

土壤传感器 A1  <---> GPIO 36 (VP)
土壤传感器 A2  <---> GPIO 39 (VN)
DHT22 数据引脚 <---> GPIO 4
继电器 IN       <---> GPIO 12
继电器 VCC      <---> VIN (5V)
继电器 GND      <---> GND
水泵 VCC        <---> 继电器 COM
水泵 GND        <---> GND
```
