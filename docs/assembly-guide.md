## 🔧 阶段一：电路面包板测试（约1小时）

### 步骤1：ESP32基础测试

```
1. 将ESP32插到面包板上
2. 用USB线连接到电脑
3. 打开Arduino IDE，选择开发板
4. 上传Blink示例程序
5. 观察板载蓝色LED是否闪烁
```

**测试代码（Arduino IDE）：**
```cpp
#define LED 2
void setup() {
  pinMode(LED, OUTPUT);
}
void loop() {
  digitalWrite(LED, HIGH);
  delay(1000);
  digitalWrite(LED, LOW);
  delay(1000);
}
```

### 步骤2：土壤传感器测试

```
1. 按接线图连接土壤传感器到GPIO34
2. 上传测试代码
3. 打开串口监视器（9600波特率）
4. 观察数值变化：
   - 空气中：2800-3200
   - 手指捏住：1500-2000
   - 插入水中：400-600
```

**测试代码：**
```cpp
const int SOIL_PIN = 34;
void setup() {
  Serial.begin(9600);
}
void loop() {
  int value = analogRead(SOIL_PIN);
  Serial.print("土壤湿度原始值: ");
  Serial.println(value);
  delay(1000);
}
```

### 步骤3：DHT22温湿度测试

```
1. 安装DHT sensor library库
2. 按接线图连接DHT22到GPIO4
3. 上传测试代码
4. 串口监视器查看温湿度
```

**测试代码：**
```cpp
#include "DHT.h"
#define DHTPIN 4
#define DHTTYPE DHT22
DHT dht(DHTPIN, DHTTYPE);
void setup() {
  Serial.begin(9600);
  dht.begin();
}
void loop() {
  float h = dht.readHumidity();
  float t = dht.readTemperature();
  Serial.print("湿度: ");
  Serial.print(h);
  Serial.print("%  温度: ");
  Serial.print(t);
  Serial.println("°C");
  delay(2000);
}
```

### 步骤4：继电器+水泵测试

```
1. 连接继电器IN1到GPIO12
2. 连接水泵到继电器NO和COM
3. 上传测试代码
4. 观察水泵是否每5秒工作一次
```

**测试代码：**
```cpp
#define PUMP_RELAY 12
void setup() {
  pinMode(PUMP_RELAY, OUTPUT);
  digitalWrite(PUMP_RELAY, HIGH); // 初始关闭
}
void loop() {
  digitalWrite(PUMP_RELAY, LOW);  // 开启水泵
  delay(5000);
  digitalWrite(PUMP_RELAY, HIGH); // 关闭水泵
  delay(5000);
}
```

## 📦 阶段二：培育箱打孔安装（约1小时）

### 步骤1：规划开孔位置

在储物箱盖子上画出开孔位置：

```

│               培育箱盖子俯视图               │

│              100x60mm                       │
│                                             │
│    [传感器孔1]        [传感器孔2]           │
│     Ø10mm              Ø10mm                │
│                                             │
│           [电源线孔]                        │
│            Ø15mm                           │
└─────────────────────────────────────────────┘
```

### 步骤2：打孔

**所需钻头：**
- Ø10mm - 土壤传感器孔 x2
- Ø20mm - 摄像头孔 x1
- Ø40mm - 风扇孔 x1
- Ø15mm - 电源线孔 x1

**打孔技巧：**
1. 用记号笔标出孔位中心
2. 用小钻头（Ø3mm）先打定位孔
3. 换大钻头扩孔
4. 用砂纸打磨孔边毛刺

### 步骤3：安装组件

**安装顺序：**

1. **ESP32-CAM摄像头**
   - 从箱内向外安装
   - 用螺丝固定或热熔胶粘合

2. **5V风扇**
   - 用4颗螺丝固定
   - 注意风向（向内吹风）

3. **LED生长灯**
   - 用双面胶粘贴
   - 或用螺丝吊装

4. **土壤传感器**
   - 从盖子插入
   - 传感器朝下，线留在箱内

## 🌱 阶段四：装土播种（约20分钟）

### 土豆切块（提前1-2天）

```
1. 选择2-3个健康发芽的土豆
2. 每个土豆切成2-3块
3. 确保每块上有1-2个芽眼
4. 切口蘸草木灰或晾干1天
5. 形成愈合层，防止腐烂
```

### 装土步骤

```
1. 营养土与珍珠岩按3:1混合
2. 每个种植盆装入约80%的土
3. 用手指在中间按一个小坑（约3cm深）
4. 放入1块土豆种薯（芽眼朝上）
5. 盖土至种植盆80%满
6. 轻轻压实，浇透水
```

### 系统启动

```
1. 将种植盆放入培育箱
2. 土壤传感器插入土中2-3cm深
3. 接通ESP32电源
4. 打开LED生长灯
5. 在Web界面设置：
   - 土壤湿度下限：30%
   - 土壤湿度上限：80%
   - 早间浇水：每天8:00，持续15秒
   - 晚间浇水：每天20:00，持续20秒
6. 启动自动监控
```

## 📊 预期时间线

| 天数 | 阶段 | 观察内容 |
|------|------|----------|
| 1-7 | 催芽期 | 土豆种块切口愈合，芽眼萌动 |
| 7-14 | 出苗期 | 土表开裂，嫩芽出土 |
| 14-30 | 生长期 | 植株长高，叶片展开 |
| 30-60 | 开花期 | 植株开花（可能不开花） |
| 60-90 | 结薯期 | 地下土豆开始膨大 |
| 90-120 | 成熟期 | 叶片变黄，可收获 |

**预期产量：** 4盆约2-3斤新土豆

### 故障处理
| 问题 | 解决方案 |
|------|----------|
| 植株发黄 | 可能浇水过多，暂停浇水2天 |
| 土壤过干 | 手动浇水一次 |
| 传感器数值异常 | 检查接线，清洁传感器 |
| 水泵不工作 | 检查继电器，是否有堵塞 |
