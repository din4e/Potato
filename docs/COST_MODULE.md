# 成本管理模块

## 📊 功能概述

成本管理模块帮助您追踪和计算土豆培育系统的所有投入成本，包括硬件采购、耗材补充、水电费用等。

## ✨ 功能特性

### 📈 数据统计
- **总投入**: 所有成本项目总计
- **分类统计**: 按类别分组显示（硬件、耗材、电费、水费、其他）
- **月度运营**: 每月运营成本（电费、水费等）
- **年度预测**: 基于月度成本计算年度总投入
- **首年总投入**: 初始硬件投入 + 一年运营成本

### 💹 图表展示
- **饼图**: 成本分类占比分布
- **进度条**: 各类别成本占比可视化
- **卡片展示**: 关键指标一目了然

### ✏️ 增删改查
- **添加**: 新增成本记录
- **编辑**: 修改现有记录
- **删除**: 移除错误记录
- **查看**: 列表展示所有成本项

## 🔌 API 接口

### 获取所有成本项
```bash
GET /api/cost/:deviceId
```

**响应示例**:
```json
{
  "success": true,
  "data": [
    {
      "id": 1,
      "deviceId": "potato-chamber-01",
      "category": "hardware",
      "name": "ESP32 开发板",
      "quantity": 1,
      "unitPrice": 18,
      "totalPrice": 18,
      "purchaseDate": "2024-01-15",
      "supplier": "淘宝"
    }
  ]
}
```

### 获取成本汇总
```bash
GET /api/cost/:deviceId/summary
```

**响应示例**:
```json
{
  "success": true,
  "data": {
    "totalCost": 110,
    "categoryBreakdown": {
      "hardware": 74,
      "supplies": 31,
      "electricity": 5,
      "water": 0,
      "other": 0
    },
    "monthlyOperating": 5,
    "projectedAnnual": 60
  }
}
```

### 添加成本项
```bash
POST /api/cost/:deviceId
Content-Type: application/json

{
  "category": "hardware",
  "name": "LED生长灯",
  "quantity": 1,
  "unitPrice": 25,
  "purchaseDate": "2024-03-03",
  "supplier": "京东",
  "notes": "全光谱补光灯"
}
```

### 更新成本项
```bash
PUT /api/cost/:deviceId/:id
Content-Type: application/json

{
  "quantity": 2,
  "unitPrice": 20
}
```

### 删除成本项
```bash
DELETE /api/cost/:deviceId/:id
```

## 📂 成本分类

| 类别 | 说明 | 示例 |
|------|------|------|
| `hardware` | 硬件设备 | ESP32、传感器、继电器、水泵 |
| `supplies` | 消耗品 | 营养土、种子、滴灌管、肥料 |
| `electricity` | 电费 | 月度电费分摊 |
| `water` | 水费 | 用水成本 |
| `other` | 其他 | 运费、工具等 |

## 🧮 成本计算公式

```
总投入 = Σ(单项成本)

单项成本 = 数量 × 单价

月度运营 = Σ(电费 + 水费)

年度预测 = 月度运营 × 12

首年总投入 = 初始硬件投入 + 年度运营
```

## 🎯 使用建议

1. **记录每次采购**: 购买硬件或耗材时及时记录
2. **定期检查**: 每月查看成本明细，了解资金流向
3. **成本优化**: 根据数据分析找出可优化的部分
4. **预算规划**: 根据成本预测合理安排后续投入
