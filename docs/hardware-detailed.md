## 详细硬件选型

### 1. ESP32 主控板

**推荐型号：ESP32-WROOM-32**

| 参数 | 规格 |
|------|------|
| 处理器 | Xtensa Dual-Core 32-bit LX6 |
| 主频 | 240MHz |
| 内存 | 520KB SRAM |
| Flash | 4MB |
| WiFi | 802.11 b/g/n |
| 蓝牙 | BLE 4.2 |
| GPIO | 30个 |
| ADC | 12位，2个SAR ADC |
| 接口 | SPI, I2C, UART, I2S |

**引脚分配建议：**

```
功能                GPIO    说明
├─ 土壤湿度传感器1   GPIO34  ADC1_CH6 (仅输入)
├─ 土壤湿度传感器2   GPIO35  ADC1_CH7 (仅输入)
├─ DHT22温湿度       GPIO4   数据引脚
├─ 水泵继电器        GPIO12  输出
├─ 风扇继电器        GPIO14  输出
├─ 状态LED           GPIO2   板载LED
├─ I2C (BH1750)      GPIO21/22 SDA/SCL
└─ UART              GPIO1/3  调试串口
```

### 2. 土壤湿度传感器

**电容式 vs 电阻式：**

| 类型 | 优点 | 缺点 | 寿命 | 价格 |
|------|------|------|------|------|
| 电容式 | 防腐蚀、精度高 | 价格略高 | 3-5年 | ¥4/个 |
| 电阻式 | 价格低廉 | 易腐蚀、不准 | 3-6月 | ¥1/个 |

> ⚠️ **强烈推荐使用电容式**，电阻式会在几个月内失效

**数据校准：**

```
空气中的值： 2800-3200 (干燥)
干燥土壤：   1500-2000
湿润土壤：   800-1200
水中：       400-600
```

### 3. 水泵选型

**微型潜水泵参数：**

| 参数 | 规格 |
|------|------|
| 工作电压 | 3.5V-5V |
| 工作电流 | 150-250mA |
| 流量 | 80-120 L/H |
| 扬程 | 0.5-1.0m |
| 吸程 | 30cm |
| 接口 | 4mm软管 |

**选型建议：**
- 小型培育箱（<50cm）：3V泵
- 中型培育箱（50-100cm）：5V泵
- 大型/多层系统：12V隔膜泵

### 4. 滴灌系统

**PE管布置：**

```
主蓄水箱 → 水泵 → 分流器
                  ├─ 主管 → 滴头1
                  ├─ 主管 → 滴头2
                  └─ 主管 → 滴头3
```

**滴头类型选择：**
- 压力补偿式：流量稳定，适合长距离
- 普通滴头：价格低，适合小型系统

## 接线方案

### 完整接线图

```
                    ESP32-WROOM-32
                    ┌─────────────┐
  USB 5V ────────────┤ VIN         │
                    │             │
                    │        GPIO34├───── 土壤传感器1 ─── GND
                    │        GPIO35├───── 土壤传感器2 ─── GND
                    │        GPIO4 ├───── DHT22 ──────────┐
                    │             │                      │
                    │        GPIO12├───── 继电器1 ────────┤
                    │             │        [水泵]        │
                    │        GPIO14├───── 继电器2 ───────┤
                    │             │        [风扇]        │
                    │             │                      │
  5V电源 ────────────┤ 5V          │                      ▼
                    │             │         DHT22
  GND ───────────────┤ GND         │        ┌─────┐
                    │             │        │DATA │
                    │        GPIO2 ├────────┤VCC  │
                    │             │        │GND  │
                    │             │        └─────┘
                    └─────────────┘
```

### 详细的继电器接线

```
ESP32              继电器模块              水泵/风扇
─────              ──────────              ──────
GPIO12 ──────────── IN1
GPIO14 ──────────── IN2
5V ──────────────── VCC
GND ─────────────── GND

继电器输出侧：
COM ─────────────── 5V电源+
NO  ─────────────── 水泵+  (常开触点)
水泵- ────────────── GND
```

### 滴灌系统连接

```
┌─────────────┐    ┌─────┐    ┌─────────────┐
│  蓄水箱     │───→│水泵 │───→│  分流器     │
│  (2L可乐瓶) │    └─────┘    └─────────────┘
└─────────────┘                    │
                                  ├─ 主管1 ─→ 滴头1
                                  ├─ 主管2 ─→ 滴头2
                                  └─ 主管3 ─→ 滴头3
```

## 安装步骤

### 第一阶段：电路连接 (30分钟)

1. **ESP32开发环境搭建**
   ```bash
   # 安装Arduino IDE
   # 添加ESP32开发板支持
   # 安装必要的库：
   - WiFi.h
   - PubSubClient (MQTT)
   - DHT sensor library
   - ESP32Camera
   ```

2. **面包板测试**
   - 按接线图连接所有组件
   - 使用面包板进行原型测试
   - 确保每个模块独立工作正常

3. **烧录测试固件**
   ```cpp
   // 测试代码框架
   - 串口输出传感器读数
   - 测试继电器开关
   - 测试WiFi连接
   ```

### 第二阶段：机械组装 (1小时)

1. **培育箱准备**
   - 在储物箱盖子上打孔：
     * 2个传感器孔（直径10mm）
     * 1个摄像头孔（直径20mm）
     * 1个风扇孔（直径40mm）
     * 2个电源线孔（直径8mm）
     * 若干透气孔（直径5mm）

2. **组件固定**
   ```
   推荐位置（俯视图）：
   ┌─────────────────────────┐
   │  [摄像头]     [风扇]     │
   │                         │
   │    [LED灯]              │
   │                         │
   │  [传感器1]   [传感器2]   │
   │                         │
   │    [种植盆1] [种植盆2]   │
   │                         │
   │  [ESP32]    [水泵]       │
   └─────────────────────────┘
   ```

3. **布线整理**
   - 使用扎带固定线材
   - 热缩管保护接头
   - 留出维护空间

### 第三阶段：软件配置 (30分钟)

1. **后端服务器配置**
   ```bash
   cd potato
   npm install
   cp .env.example .env
   # 编辑.env配置MQTT和数据库
   npm run dev
   ```

2. **ESP32固件配置**
   ```cpp
   // 修改main.cpp中的配置：
   const char* WIFI_SSID = "你的WiFi";
   const char* WIFI_PASSWORD = "密码";
   const char* MQTT_SERVER = "服务器IP";
   ```

3. **联调测试**
   - 确认ESP32连接到MQTT
   - 确认传感器数据正常上报
   - 测试远程控制水泵/风扇

### 第四阶段：种植准备 (20分钟)

1. **土壤准备**
   - 混合营养土和珍珠岩（3:1）
   - 保持湿度约60%
   - 装入种植盆

2. **土豆播种**
   - 选择发芽的健康土豆
   - 切块确保每块有芽眼
   - 埋入土中3-5cm

3. **系统初始化**
   - 设置浇水阈值
   - 配置定时任务
   - 启动自动监控

## 故障排查

### 常见问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| ESP32无法连接WiFi | 密码错误/信号弱 | 检查SSID和密码 |
| 土壤传感器读数不变 | 接线松动/损坏 | 检查杜邦线连接 |
| 水泵不工作 | 继电器故障/供电不足 | 测试继电器独立供电 |
| DHT22读数为NaN | 时序错误/接线反了 | 检查数据线上拉电阻 |
| MQTT连接失败 | IP错误/防火墙 | ping测试MQTT服务器 |
