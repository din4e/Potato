## 🔌 ESP32 引脚分配表

### 核心引脚定义

| 功能 | GPIO | 说明 | 备注 |
|------|------|------|------|
| 土壤湿度传感器1 | GPIO34 | ADC1_CH6 | 仅输入，推荐使用 |
| 土壤湿度传感器2 | GPIO35 | ADC1_CH7 | 仅输入，推荐使用 |
| DHT22温湿度 | GPIO4 | 数据引脚 | 需要上拉电阻 |
| 水泵继电器 | GPIO12 | 输出控制 | 低电平触发 |
| 风扇继电器 | GPIO14 | 输出控制 | 低电平触发 |
| 状态LED | GPIO2 | 板载LED | 调试用 |
| I2C SDA | GPIO21 | 扩展用 | BH1750/OLED |
| I2C SCL | GPIO22 | 扩展用 | BH1750/OLED |
| TX0 | GPIO1 | UART发送 | 调试串口 |
| RX0 | GPIO3 | UART接收 | 调试串口 |

### 电源引脚

| 引脚 | 说明 | 连接 |
|------|------|------|
| VIN | 7-12V输入 | 可接电池或外部电源 |
| 3V3 | 3.3V输出 | 给部分传感器供电 |
| GND | 地线 | 所有GND连接在一起 |
| EN | 复位使能 | 接3V3或悬空 |

## 📌 逐个模块接线

### 1. 土壤湿度传感器（电容式）

**传感器1 (GPIO34)**
```
传感器     →    ESP32
─────────────────────────
VCC (红)  →    3.3V
GND (黑)  →    GND
SIG (白)  →    GPIO34 (ADC1_CH6)
```

**传感器2 (GPIO35)**
```
传感器     →    ESP32
─────────────────────────
VCC (红)  →    3.3V
GND (黑)  →    GND
SIG (白)  →    GPIO35 (ADC1_CH7)
```

> ⚠️ 注意：电容式传感器支持3.3V供电，电阻式必须用5V

### 2. DHT22 温湿度传感器

```
DHT22     →    ESP32
─────────────────────────
VCC (红)  →    3.3V
GND (黑)  →    GND
DATA (黄) →    GPIO4
```

> 💡 如果传感器不稳定，在DATA和VCC之间加一个10KΩ上拉电阻

### 3. 继电器模块（控制水泵）

**继电器1 - 水泵 (GPIO12)**
```
ESP32     →    继电器      →    水泵
───────────────────────────────────
GPIO12    →    IN1
5V        →    VCC
GND       →    GND

          →    COM (公共端) → 5V电源正极
          →    NO (常开)   → 水泵正极(红线)
水泵负极(黑线) → GND
```

**继电器2 - 风扇 (GPIO14)**
```
ESP32     →    继电器      →    风扇
───────────────────────────────────
GPIO14    →    IN2
5V        →    VCC
GND       →    GND

          →    COM (公共端) → 5V电源正极
          →    NO (常开)   → 风扇正极(红线)
风扇负极(黑线) → GND
```

> 📌 继电器是**低电平触发**：GPIO输出LOW时，水泵开启

### 4. ESP32-CAM 摄像头模块

**ESP32-CAM 接线（独立供电）**
```
ESP32-CAM 引脚分配：
─────────────────────────────────
VCC       →    5V (独立供电)
GND       →    GND
U0R       →    ESP32 GPIO16 (可选)
U0T       →    ESP32 GPIO17 (可选)
```

**USB转TTL下载器接线（烧录固件用）**
```
下载器      →    ESP32-CAM
─────────────────────────
5V        →    5V
GND       →    GND
TX        →    U0R
RX        →    U0T
GPIO0     →    GND (烧录时接地)
```

### 5. LED生长灯

```
LED灯板    →    供电
─────────────────────────
VCC/ +    →    12V (需单独适配器)
GND/ -    →    GND
```

> 💡 生长灯功率较大(通常5-10W)，建议独立供电

### 6. 滴灌系统

```
水箱 → 水泵 → PE主管 → 分流器
                      ├─ 支管1 → 滴头1 → 种植盆1
                      ├─ 支管2 → 滴头2 → 种植盆2
                      └─ 支管3 → 滴头3 → 种植盆3
```

## ✅ 接线检查清单

### 上电前检查

- [ ] 所有VCC连接正确（未接反）
- [ ] 所有GND连接在一起（共地）
- [ ] 继电器控制脚连接到正确GPIO
- [ ] 水泵极性正确（红线接正极）
- [ ] 传感器引脚顺序正确
- [ ] 无短路（用万用表测试VCC-GND阻值）
- [ ] ESP32-CAM的GPIO0已悬空（非烧录状态）

### 初次上电测试

1. **仅接USB供电**（不接外设）
   - [ ] 板载LED闪烁
   - [ ] 串口有输出

2. **接传感器**
   - [ ] 土壤传感器读数变化
   - [ ] DHT22正常返回数值

3. **接继电器**
   - [ ] 继电器指示灯随命令变化
   - [ ] 继电器有"咔嗒"声

4. **接执行器**
   - [ ] 水泵正常启停
   - [ ] 风扇正常转动

## 🐛 常见接线问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| ESP32无法启动 | 供电不足/短路 | 检查VCC-GND是否短路 |
| 传感器读数为0 | 接线松动/供电错误 | 重新检查杜邦线 |
| 继电器不动作 | GPIO错误/触发方式错误 | 检查GPIO号和触发逻辑 |
| DHT22返回NaN | 时序错误/接线反了 | 检查DATA引脚和上拉电阻 |
| 水泵不工作 | 继电器接线错误/水泵损坏 | 用万用表测水泵两端电压 |
